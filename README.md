# 📝 CLI Todo Application

Консольное приложение для управления задачами, разработанное в рамках курса "Технологии программирования".

## 🚀 Возможности

- ✅ **Добавление задач** с названием, описанием и приоритетом
- 📋 **Просмотр задач** с фильтрацией (все/активные/выполненные)
- ✓ **Отметка выполнения** с сохранением времени завершения
- 🗑️ **Удаление задач**
- 💾 **Сохранение данных** между запусками
- 🎨 **Удобный интерфейс** с emoji и цветовым оформлением
- 🔧 **Гибкая настройка** приоритетов и фильтров

## 📦 Установка

### Требования
- Java 11 или выше
- Apache Maven 3.6+

### Сборка из исходного кода

```powershell
# Клонирование (если используется Git)
git clone <репозиторий>
cd cli-todo-app

# Сборка проекта
mvn clean package

# Запуск
java -jar target/cli-todo-1.0-SNAPSHOT.jar --help

# Из папки проекта
.\start-todo.bat

# Или через PowerShell
todo --help

📖 Использование
Основные команды
bash
# Получить справку
todo --help

# Добавить задачу
todo add "Купить продукты" -d "Молоко, хлеб, яйца" -p HIGH

# Посмотреть все задачи
todo list

# Посмотреть только активные задачи
todo list -todo

# Посмотреть только выполненные задачи
todo list -completed

# Отметить задачу как выполненную
todo complete 1

# Удалить задачу
todo delete 2

# Инициализировать примерами (если список пуст)
todo init

# Очистить все задачи
todo clear

Приоритеты задач
Задачи могут иметь три уровня приоритета:

LOW (⬇) - Низкий приоритет

MEDIUM (➡) - Средний приоритет (по умолчанию)

HIGH (⬆) - Высокий приоритет

🏗️ Архитектура проекта
Проект следует многослойной архитектуре:

1. Модель (Model)
Класс Task представляет задачу со следующими полями:

id - уникальный идентификатор

title - название задачи

description - описание

completed - статус выполнения

priority - приоритет (LOW, MEDIUM, HIGH)

createdAt, updatedAt, completedAt - временные метки

2. Репозиторий (Repository)
TaskRepository отвечает за:

Сохранение/загрузку задач из файла todo-data.ser

Управление уникальными идентификаторами

CRUD операции над задачами

3. Сервис (Service)
TaskService содержит бизнес-логику:

Валидация данных

Обработка исключений

Преобразование данных

4. CLI Слой
Команды Picocli для взаимодействия с пользователем:

TodoApp - главный класс

AddCommand, ListCommand, CompleteCommand, DeleteCommand - команды

📊 CI/CD
Проект использует GitHub Actions для непрерывной интеграции:

✅ Автоматическая сборка при пуше в main или develop

✅ Проверка стиля кода с Checkstyle

✅ Запуск unit-тестов

✅ Создание JAR артефакта
